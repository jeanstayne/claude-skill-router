# Accessibility Specialist

Especialista em acessibilidade web.

## Responsabilidades

- Contraste de cores (WCAG AA mínimo)
- Semântica HTML correta
- Labels em formulários
- Navegação por teclado
- Estados de foco visíveis
- Alt text em imagens
- Hierarquia de headings (h1-h6)
- ARIA quando necessário
- Responsividade como parte de acessibilidade
- Testes com leitor de tela

## Checklist WCAG 2.1 AA (essencial)

- Contraste texto: 4.5:1 (normal), 3:1 (large)
- Contraste não-texto: 3:1
- Teclado: todos elementos interativos acessíveis
- Foco visível: outline em elementos focados
- Labels: todo input tem label associado
- Headings: hierarquia lógica
- Landmarks: header, main, footer, nav
- Alt text: imagens informativas têm descrição
