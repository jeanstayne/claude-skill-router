# Framer Marketing Site Director

## Objetivo

Criar sites de marketing e landing pages com estética editorial e comercial premium. Foco em hero sections impactantes, animações suaves, storytelling visual e ritmo de página que guia o usuário.

## Quando usar

- Site de marketing de produto
- Landing page de lançamento
- Site institucional com narrativa visual
- Página de vendas com storytelling
- Refatorar site existente com visual datado

## Quando não usar

- Dashboard ou sistema interno com foco em dados
- Aplicação web com lógica de negócio complexa
- Projeto puramente técnico sem foco em marketing

## Entradas esperadas

- Produto/serviço a ser apresentado
- Público-alvo e persona
- Tom de comunicação
- Referências visuais de sites similares
- Assets disponíveis (imagens, vídeos, ilustrações)

## Processo

1. Definir narrativa e jornada do usuário
2. Estruturar seções em ordem de storytelling
3. Criar hero section com hierarquia visual forte
4. Desenvolver seções com ritmo visual
5. Adicionar microinterações e animações
6. Garantir que cada seção tem propósito claro
7. Revisar fluxo de leitura e escaneabilidade

## Checklist

- Hero com headline + subheadline + CTA + visual
- Seções com ritmo visual (alternância de densidade)
- Animações sutis (scroll, hover, entrada)
- Storytelling visual (não apenas texto)
- CTA visível em momentos estratégicos
- Imagens e ilustrações de alta qualidade
- Tipografia com personalidade
- Responsivo sem perder impacto visual
- Performance (imagens otimizadas, animações leves)

## Formato de saída

```md
## Narrativa
[Jornada do usuário na página]

## Estrutura de seções
1. Hero — [objetivo]
2. [Seção] — [objetivo]
3. ...

## Direção visual
[Estilo editorial, atmosfera, referências]

## Animações
[Tipo de animação por seção]
```

## Regras de qualidade

- Cada seção tem um propósito claro
- Hero ocupa viewport inteira no primeiro impacto
- Animações são sutis, não distractivas
- Texto é escaneável (headings, bullets, destaque)
- Imagens são otimizadas (WebP, lazy load)

## Não fazer

- Não usar animações excessivas ou distractivas
- Não criar seções sem propósito
- Não usar imagens de baixa qualidade
- Não esquecer performance
- Não criar hero genérico
- Não usar Lorem Ipsum
