# Brand Visual Director

## Objetivo

Definir e aplicar direção visual única para cada marca, garantindo diferenciação estética e consistência.

## Quando usar

- Criar landpage para marca com identidade definida
- Landing page para marca nova (sem guia visual)
- Refatorar layout genérico

## Quando não usar

- Projetos puramente técnicos sem interface visual
- Sistemas que já possuem design system consolidado

## Processo

1. Analisar referências visuais da marca
2. Extrair paleta de cores principal
3. Definir tipografia (headings e body)
4. Criar sistema de botões (primário, secundário, ghost)
5. Definir estilo de cards
6. Escolher tratamento de imagens
7. Definir densidade visual (espaçamento, respiro)
8. Aplicar consistência em todas as seções

## O que evitar

- Layout genérico "template Tailwind"
- Falta de diferenciação entre marcas
- Cores sem contraste adequado
- Tipografia padrão do navegador
- Botões sem hierarquia clara
- Cards sem identidade
- Excesso de informação visual
- Uso indevido de gradientes ou sombras

```md

[Resumo da personalidade visual]

[3-5 cores com HEX]

- Headings: [fonte]
- Body: [fonte]

## Componentes
[Estilo de botões, cards, inputs]

## Referências
[Imagens ou links de inspiração]
```
